Placeholders

Placeholders represent content that is waiting to be loaded, when the content request is complete the placeholder disappears and the content appears in its place. To users, this makes the content seemingly load faster as well as shows them where the content will be rendered.

Example

Register the Placeholder directive in your component. Then add the markup.

  import {Placeholder} from 'clarity-angular/placeholder/placeholder;

  @Component({
      ...
      directives: [Placeholder],
      ...
  });
  export class YourComponent{

  }

Add directive to template

<clr-placeholder clr-type="paragraph" ></clr-placeholder>
