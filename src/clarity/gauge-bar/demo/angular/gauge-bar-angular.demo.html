Angular Component

Clarity bar gauges.

Summary of Options

Input Values Default Effect
[clrBarGaugeValueOne] integer 0 <= x <= 100 0 -
[clrBarGaugeValueTwo] integer 0 <= x <= 100 0 Will only be visible if valueTwo > valueOne.
[clrBarGaugeColorOne] string rgb(0, 106, 144) -
[clrBarGaugeColorOne] string rgb(157, 197, 217) -
[clrBarGaugeBackgroundColor] string rgb(224, 224, 224) The background color of the remainder of the bar.
[clrBarGaugeAnimate] boolean true Whether or not the gauge bars should animate when values change
