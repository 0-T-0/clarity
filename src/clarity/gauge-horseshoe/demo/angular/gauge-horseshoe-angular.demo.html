Angular Component

Clarity Horseshoe gauges are easily customized using CSS.

Summary of Options

Input Values Default Effect
[clrHsGaugeValueOne] integer 0 <= x <= 100 0 -
[clrHsGaugeValueTwo] integer 0 <= x <= 100 0 Will only be visible if valueTwo > valueOne.
[clrHsGaugeColorOne] string rgb(0, 106, 144) -
[clrHsGaugeColorOne] string rgb(157, 197, 217) -
[clrHsGaugeBackgroundColor] string rgb(255, 255, 255) The background color of the central part of the gauge.
[clrHsGaugeAnimate] boolean true Whether or not the gauge bars should animate when values change
