Angular Component

Clarity gauges are available in three sizes by default and are easily customized.

Summary of Options

Input Values Default Effect
[clrGaugeValueOne] integer 0 <= x <= 100 0 -
[clrGaugeValueTwo] integer 0 <= x <= 100 0 Will only be visible if valueTwo > valueOne.
[clrGaugeColorOne] string rgb(0, 106, 144) -
[clrGaugeColorOne] string rgb(157, 197, 217) -
[clrGaugeBackgroundColor] string rgb(255, 255, 255) The background color of the central part of the gauge.
[clrGaugeAnimate] boolean true Whether or not the gauge bars should animate when values change
